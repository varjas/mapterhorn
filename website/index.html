About

Mapterhorn is an open-data project that was made possible thanks to support from the NGI0 Core Fund, a fund established by NLnet with financial support from the European Commission's Next Generation Internet programme.

Mapterhorn is currently maintained by Oliver Wipfli. He shares admin access rights to the GitHub organization and the mapterhorn.com domain with Brandon Liu and Sascha Brawer.
